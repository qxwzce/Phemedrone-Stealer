[EN]
1. Upload gate.php on your server and set 777 rights
2. Edit bot token and chat id in gate.php
3. Create "logs" folder next to the php file
4. Set methode Gate in stealer and write gate url (example: https://domain/gate.php)

[RU]
1. Загрузить файл gate.php на ваш сервер
2. Поменять значения bot token и chat id на ваше
3. Создать папку "logs" рядом с php файлом
4. Поставить метод Gate в стиллере и написать ссылку на гейт (пример: https://domain/gate.php)
